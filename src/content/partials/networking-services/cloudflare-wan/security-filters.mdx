Cloudflare WAN (formerly Magic WAN) customers have <a href="/cloudflare-network-firewall/plans/">automatic access to Cloudflare Network Firewall</a>. Cloudflare Network Firewall is Cloudflare's firewall-as-a-service solution that allows you to protect your infrastructure. Cloudflare Network Firewall supports layers three and four of the [OSI model](https://www.cloudflare.com/learning/ddos/glossary/open-systems-interconnection-model-osi/), and enables you to allow or block traffic on a variety of packet characteristics.

Refer to <a href="/cloudflare-network-firewall/">Cloudflare Network Firewall</a> for more information about this product.

As a Cloudflare WAN customer, you can also use Cloudflare Gateway to set up policies to inspect network and HTTP traffic to the Internet or your private network infrastructure. Refer to <a href={props.gatewayURL}>Connect to Cloudflare Gateway with Cloudflare WAN</a> to learn how to filter Cloudflare WAN traffic with Gateway policies.
